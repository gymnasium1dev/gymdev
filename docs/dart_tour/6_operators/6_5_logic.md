# Логические операторы

Вы можете инвертировать или комбинировать логические выражения, используя логические операторы.

| Оператор | Описание                            |
| -------- | ----------------------------------- |
| !expr    | инверсия значения                   |
| ||       | логическое ИЛИ                      |
| &&       | логическое И                        |
