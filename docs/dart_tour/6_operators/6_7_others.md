# Другие операторы

| Оператор | Описание                            |
| -------- | ----------------------------------- |
| () | Вызов функции(иногда соправождается передачей аргументов в функцию) |
| [] | Определение массива                      |
| {} | Определение славоря                      |
| .  | Вызов методов                            |
| ?. | Вызов методов с проверкой на null        |
